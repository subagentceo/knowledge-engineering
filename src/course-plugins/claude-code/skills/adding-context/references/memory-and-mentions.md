# Injecting context: `#` memory and `@` file mention

Two inline shortcuts for editing context files and pulling specific files into a request.

## `#` memory shortcut

Type a request starting with `#` to have Claude edit the appropriate CLAUDE.md file for you in natural language, instead of editing it by hand.

## `@` file mention

Prefix a path with `@` to pull that exact file into the request as targeted context. Use it when you already know which file matters, instead of paying for Claude to search.
