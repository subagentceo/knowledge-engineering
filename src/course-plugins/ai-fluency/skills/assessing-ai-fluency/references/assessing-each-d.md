# Assessing each D

- **Delegation** — *Outcome:* did the plan make sense, goals realistic, right tool? *Process:* annotated chat logs showing option exploration. *Reflection:* explain choices, alternatives, how delegation shaped the result.
- **Description** — *Outcome:* prompt/response quality and evolution from first to final. *Process:* logs showing iterative refinement and shared-context building. *Reflection:* which techniques worked, task-specific approaches.
- **Discernment** — *Outcome:* annotations marking strengths/weaknesses; decision logs of kept/modified/discarded. *Process:* in-line evaluation, error catching, pattern recognition. *Reflection:* how criteria evolved, initially missed issues.
- **Diligence** — *Outcome:* diligence statements, attribution/transparency docs, fact-checking evidence. *Process:* data handling, ethical decisions, constraint checking. *Reflection:* ethical dilemmas, responsibility, values.
