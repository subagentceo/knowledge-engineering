# The four assessment matrices

Each D has a matrix defining strong performance plus outcome/process/reflection focus areas:

- **Delegation** — strategic thinking about when/why/how to collaborate (task division, tool selection, goal achievement; decision documentation; choice analysis).
- **Description** — effective communication about outputs/processes/behaviors (prompt evolution, context building; iterative refinement, shared vocabulary; strategy analysis).
- **Discernment** — critical evaluation with domain-specific, nuanced criteria (annotation quality, evidence-based judgment; error catching, improvement over time; criteria evolution).
- **Diligence** — integrated ethical reasoning throughout (attribution/verification quality; data handling, constraint checking; ethical-dilemma discussion, values articulation).

Use matrices to develop rubrics (select relevant Ds, pick assessment types, adapt indicators, define performance levels), guide feedback, and share criteria with students beforehand so they self-identify evidence.
