# Diligence as field-values embodiment (disciplines)

- **Codify ethical frameworks** — define your field's "do no harm"; build case studies on ethical edges; create decision matrices.
- **Clarify transparency norms** — document disclosure expectations by field; create AI-assistance disclosure/citation templates.
- **Co-create accountability policies** — build standards *with* students rather than imposing them; develop peer-review protocols.
