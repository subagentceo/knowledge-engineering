# Four ways to teach the framework

1. **Step-by-step (linear)** — Delegation → Description → Discernment → Diligence as a sequence. Best for beginners; four-part sequential assignments.
2. **Start anywhere (non-linear)** — any competency is a valid entry point; they inform each other. Best for experienced users; group/case-based work.
3. **Just one D (focused)** — deep dive on a single competency (e.g., three Discernment workshops on Product/Process/Performance). Best for skill-building sessions; avoids overload.
4. **Two loops** — teach the Delegation–Diligence and Description–Discernment loops as nested processes. Most conceptually rich; for students who already have the basics.

## Reusable context document

Both the educator and student courses recommend building a reusable **context document** (Teaching Context Document for educators; Learning Context Document for students) that establishes shared understanding with AI — subject/major, goals, constraints, boundaries, collaboration preferences — so future sessions start with rich context instead of from scratch.
