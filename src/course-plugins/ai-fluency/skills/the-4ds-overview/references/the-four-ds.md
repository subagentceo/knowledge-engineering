# The Four Ds

AI Fluency is captured by four interconnected competencies — the **4Ds** — that together create fluency:

- **Delegation** — determining what work should be done by humans, by AI, or together.
- **Description** — how you communicate with AI: a collaborative conversation, not just commands.
- **Discernment** — recognizing good vs bad results and adapting accordingly.
- **Diligence** — taking responsibility for AI use through ethical, transparent, accountable practice.

The 4Ds are not a checklist; they are interconnected competencies that work together and inform each other. Because they focus on strategic thinking, communication, evaluation, and ethical responsibility — not on any specific tool — the skills stay relevant as the technology evolves.

## Each D has three sub-competencies

- **Delegation** = Problem Awareness · Platform Awareness · Task Delegation
- **Description** = Product · Process · Performance description
- **Discernment** = Product · Process · Performance discernment
- **Diligence** = Creation · Transparency · Deployment diligence
