# Default behavior is the problem to overcome

AI is built to give **direct answers**, which is often unhelpful for *learning*. Reconfigure it with description:

- "Ask me questions instead of explaining."
- "Point me toward the solution without solving it."
- "Help me dig deeper into my own interpretation."

Useful learning-oriented outputs you can describe for: pointers/clues for hard topics, comprehension-testing questions, critical feedback from a different perspective, re-explaining a concept with new examples, progressive practice problems, or having the AI act as a student so you teach it.
