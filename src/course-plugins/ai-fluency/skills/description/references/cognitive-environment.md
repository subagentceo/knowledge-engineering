# Building a cognitive environment (Teaching AI Fluency)

Description's real craft is constructing a shared **cognitive environment**: shared vocabulary and references that evolve over time, well-defined goals/values/methods, established interaction patterns, and mechanisms to build on previous turns. Three lenses on it: **Product** (what you're creating together), **Process** (shared ways of thinking), **Performance** (the human–AI relationship dynamics). Success shows up as **shared shorthand language** and an **exploration mindset** (flexible interaction replacing rigid commands).
