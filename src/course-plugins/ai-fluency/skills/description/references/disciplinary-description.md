# Description in disciplinary/teaching work

- *Map discipline products* — document key field outputs precisely; have students reverse-engineer professional outputs.
- *Reveal expert thinking* — make problem-solving processes visible; trace micro-decisions; build flowcharts of expert thinking.
- *Name field norms* — surface the behaviors that define the field; define "thinking like an X" in concrete terms.
- For educators building materials: **leverage established context** (prior conversations, the teaching context document, shared transcripts) plus the specific requirements for this artifact.
