# Description

> Distilled from the *AI Fluency for Educators / Students* and *Teaching AI Fluency* courses.

## Overview

**Description = how you communicate with AI.** It is a collaborative *conversation*, not just prompting — you build shared context and understanding over multiple interactions, not single prompt exchanges. Description never works alone: it runs in the **Description–Discernment loop** (describe a need → evaluate the result → refine the description, iteratively). See the `discernment` skill.

- [Three dimensions of description](references/dimensions.md) — Product, Process, and Performance description.
- [Learning-oriented description](references/learning-oriented-description.md) — overcoming the direct-answer default; configuring AI to teach.
- [Building a cognitive environment](references/cognitive-environment.md) — the shared vocabulary, goals, methods, and interaction patterns the craft really builds.
- [Description in disciplinary/teaching work](references/disciplinary-description.md) — mapping products, revealing expert thinking, naming field norms, leveraging context.
