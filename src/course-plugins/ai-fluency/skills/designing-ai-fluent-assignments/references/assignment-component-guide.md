# Assignment component guide

Pick components by the competency you want to develop or assess.

**Outcome-based** (what students produce):
- *Improving AI outputs* — transform mediocre AI output into excellent work. Focus: Discernment, Description.
- *Product comparison* — same task on multiple AI systems + comparison matrix + recommendation. Focus: Delegation, Discernment.
- *Constraint-based challenges* — hit specific format/length/style/audience (or ethical/conflicting/time) constraints. Focus: Description, Delegation.
- *Peer product review* — students set goals, build with AI, swap and critique against goals and ethics. Focus: Discernment, Diligence.

**Process-based** (how students work over time):
- *Annotated chat logs* — submit logs marking turning points, breakthroughs, failures/recovery, communication evolution, shared-context development. Focus: Description, Discernment.
- *Recorded narrations* — screen recording with real-time narration of choices, evaluation, alternatives, ethics. Focus: all 4Ds.
- *Process playbooks* — personal AI-strategy guides (conversation guides, decision trees, checklists, recovery, ethics). Focus: all 4Ds.
- *AI-/peer-assisted debrief* — structured analysis of a collaboration with an external perspective. Focus: all 4Ds.

**Reflection-based** (metacognition):
- *Guided inquiry* — targeted questions (surprises, failed strategies, ethical decisions).
- *Learning journal* — ongoing weekly entries + monthly synthesis of 4D development.
- *Scenarios / case studies* — apply learning to realistic situations. Focus: Diligence, Delegation.
- *Personal policy statements* — synthesize values, criteria, standards, boundaries into a personal AI framework. Focus: Diligence, all 4Ds.
