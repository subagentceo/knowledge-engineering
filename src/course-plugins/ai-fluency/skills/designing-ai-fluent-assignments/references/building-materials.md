# Building materials (the Description–Discernment loop)

Established context makes each new artifact better than starting fresh. For any material: **set up** (share course context + specifics) → **process** (outline → evaluate → develop → assemble) → **final check** (verify accuracy, flag image/citation needs, document AI contribution). Four material types from the course: lecture slide deck, study/revision guide, interactive in-class exercise, knowledge-check quiz.
