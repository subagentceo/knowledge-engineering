# Model selection

Three families, each optimized for a different priority:

- **Opus** — highest intelligence for complex, multi-step reasoning and planning. Higher cost and latency.
- **Sonnet** — balanced intelligence/speed/cost; strong coding and precise code edits. Best default for most use cases.
- **Haiku** — fastest and cheapest; best for real-time interactions and high-volume processing.

Selection rule: intelligence-priority → Opus, speed-priority → Haiku, balanced → Sonnet. It is common to use several models in one app, choosing per task. All families share text generation, coding, and image analysis.
