# claude-cowork

A Claude Code plugin distilling the **Introduction to Claude Cowork** course into actionable skills. Cowork is the mode of Claude inside the desktop app where Claude works *in your environment* — on your files, in your apps, with your tools — to take whole pieces of work off your plate and return a real deliverable.

The skills here cover knowing when to reach for Cowork, getting it set up, delegating and steering tasks, automating them, and building the standing context (global instructions, projects, skills, plugins) that makes Cowork better at your work over time. This is the Cowork product surface — guidance is distilled entirely from the course transcript.

## Source course

`projects/courses/introduction-to-claude-cowork__cowork.txt` — *Introduction to Claude Cowork* (Anthropic). Lessons 1–13.

## Skills

| Skill | What it covers |
|---|---|
| **what-is-cowork** | What Cowork is; the delegating-not-chatting shift; Chat vs. Cowork vs. Code. |
| **match-task-to-mode** | Picking Chat/Cowork/Code; the three patterns (multi-step, file-based, multi-tool) that fit Cowork. |
| **setup-and-folder-scoping** | Installing Cowork; pointing Claude at the smallest right folder; read/write scope; cloud-file behavior. |
| **connectors** | Setting up and toggling connectors per task; which to add first; Claude in Chrome as the no-connector bridge. |
| **permissions-and-safe-use** | The approval mode selector; the delete guarantee; safe-setup, prompt, and in-the-moment checks; when *not* to use Cowork. |
| **delegate-a-task** | Writing a prompt that names deliverable/inputs/nuance; clarifying questions; steering mid-task; reviewing the deliverable. |
| **scheduled-tasks-and-dispatch** | `/schedule` cadences and Dispatch from your phone; the on-and-awake constraint; availability. |
| **global-instructions-and-projects** | Standing context: global instructions about you, and projects (instructions, scheduled tasks, context, auto-built memory). |
| **build-skills-and-plugins** | Building skills (instructions/assets/references/scripts), validating with evals, bundling into plugins, sharing across a team. |
| **chrome-and-m365-surfaces** | Claude in Chrome for browser tools with no connector; Claude inside Word/Excel/PowerPoint/Outlook; Cowork vs. M365. |
