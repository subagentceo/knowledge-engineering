# Build Skills and Plugins for Cowork

> Distilled from the *Introduction to Claude Cowork* course.

## Overview

The signal you're ready for a skill: **you're explaining the same workflow to Claude multiple times.** Skills, projects, global instructions, and plugins are independent and compound — the more Cowork knows, the more it takes off your plate. The arc: build a skill, validate it with evals, bundle skills into a plugin, then share it across your team.

- [Skills](references/skills.md) — what a skill is, what goes inside one (instructions, assets, references, scripts), and how to build one with Claude.
- [Validate with evals](references/validate-with-evals.md) — using skill-creator's built-in with-skill vs. without-skill pairs to test before you share.
- [Plugins](references/plugins.md) — bundling skills around a job, the two shapes, and Anthropic's off-the-shelf role plugins.
- [Share across a team](references/share-across-a-team.md) — distributing via a private marketplace, install levels, and habits worth keeping.
