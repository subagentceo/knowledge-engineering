# Plugins — Bundle Skills Around a Job

A plugin is a packaged set of skills (plus the connectors and subagents they depend on) built around a job. Two shapes:
- **End-to-end process** — skills for each sequential step of one workflow (e.g. monthly-close: pull actuals → variance table → board memo).
- **Most-used skills bundled** — independent skills a team reaches for most (e.g. finance: variance analysis, modeling, investment memos, quarterly reports).

Anthropic publishes plugins for common roles (finance, legal, sales, marketing, support, PM). Find them in **Customize → Plugins**: install off the shelf, **customize** (Customize opens a Cowork task where you tailor it to your templates/definitions — *"here are our last 3 NDAs, update the /nda-triage skill to match"*), or build your own. Try `/setup-cowork` for a short interview that suggests a fitting plugin.
