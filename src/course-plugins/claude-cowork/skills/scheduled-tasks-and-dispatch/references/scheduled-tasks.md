# Scheduled Tasks — Work on a Cadence

Set a task up once and have Claude run it on a cadence. Two ways:

- **From scratch:** type `/schedule`, describe the task and cadence, and Claude drafts a prompt for you to review and accept.
- **From a proven run (the more natural path):** do the task once in Cowork, confirm the output looks right, then type `/schedule` to turn that exact process into a recurring one.

Tell Claude your cadence — **hourly, daily, weekdays, or manual** — review the prompt, and accept.

**Constraint:** scheduled tasks only run when your computer is **on and Claude is running**. If the laptop is closed, asleep, or off at the scheduled time, Claude picks the task up as soon as you're back and tells you it was delayed.

Ideas worth scheduling:
- A Friday review: what shipped this week, a recap of major decisions in Slack/Teams, a preview of next week.
- A monthly metrics roll-up pulled from your spreadsheets on the 1st of each month.
- A morning briefing for the day's calendar that draws on past meeting notes and email threads for context.

(Scheduled tasks can also belong to a **project** — set up from a conversation inside it, they run with the project's context every time.)
