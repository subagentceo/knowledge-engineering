# Scheduled Tasks and Dispatch

> Distilled from the *Introduction to Claude Cowork* course.

## Overview

These cover *when* and *where* a handoff happens: tasks Claude starts on a schedule, and tasks you start from your phone. Both only run when your desktop is on, awake, and signed in with Cowork open.

- [Scheduled tasks — work on a cadence](references/scheduled-tasks.md) — set up with `/schedule` (from scratch or from a proven run), cadences, the computer-on constraint, and ideas worth scheduling.
- [Dispatch — start a task from your phone](references/dispatch.md) — assign from the mobile app, run on the desktop, availability, and the safety note that applies to anything running while you're not watching.
