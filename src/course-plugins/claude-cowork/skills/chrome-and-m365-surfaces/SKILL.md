# Cowork Beyond the Desktop: Chrome and Microsoft 365

> Distilled from the *Introduction to Claude Cowork* course.

## Overview

Cowork extends past the desktop folder onto two more surfaces: the browser, for any tool with no connector, and Microsoft 365, for working inside Office files in place. Both stay in one conversation with Cowork and hand off cleanly — the file is the file everywhere.

- [Claude in Chrome](references/claude-in-chrome.md) — the bridge for tools with no connector: dashboards, portals, and web apps behind a login, plus the control and watch-outs.
- [Claude in Microsoft 365](references/claude-in-m365.md) — the add-in inside Word, Excel, PowerPoint, and Outlook, cross-app moves, and when to use Cowork vs. Claude in M365.
