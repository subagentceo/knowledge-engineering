# Permissions Model and Safe Use

> Distilled from the *Introduction to Claude Cowork* course.

## Overview

Cowork is an autonomous tool with file, app, and connector access — so two things matter: the approval modes that govern what Claude does on its own, and the habits that keep mistakes away from what's irreplaceable. Start from the permissions model, then layer the safe-use practices on top.

- [The permissions model (approvals)](references/permissions-model.md) — the mode selector (Ask before acting vs. Act without asking), the unskippable delete confirmation, and the connector/web-access controls you also hold.
- [Best practices for working safely](references/safe-use.md) — folder-as-boundary setup, prompts that leave no room for the wrong action, the three in-the-moment checks, and when Cowork isn't the right tool.
