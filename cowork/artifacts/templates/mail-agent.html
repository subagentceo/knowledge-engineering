Mail

agent application · e2m-nativepreview
Local preview of the agent-native mail template, implemented over our e2m protocol — no npx, no live fetch. Mapping: an email = an Envelope, a reply = a Transition, the inbox = the e2m mailbox (cowork/data/mailbox/<agent>.jsonl). Sample envelopes embedded inline.
e2m ↔ mail: from/to = sender/recipient · subject = subject · envelope_type = label (task/result/escalate/operator/notify) · payload = structured attachment · state = read/unread · actions write back to the JSONL as transitions (preview only).
