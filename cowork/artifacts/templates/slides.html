pptx generation from structured content — turns product_vision.html sections and weekly metrics into a slide deck.

description

Slides takes any cowork/ artifact (connector_audit_94.html, priority_tasks.html, metrics scorecard) and dispatches a task to the design queue to generate a .pptx via the pptx skill. The envelope carries the source artifact path and a slide outline.

dispatch task via e2m-mcp

Write an envelope to cowork/data/queues/design.jsonl → agent design-agent picks it up from mailbox.






e2m-mcp tools used by this template



  
envelope_write

  
Validates and appends a DurableTask envelope to cowork/data/queues/design.jsonl.





  
mailbox_send

  
Routes the task notification to cowork/data/mailbox/design-agent.jsonl with type=task.





  
task_transition

  
Appends state transitions (claim → complete) as the agent works through the task.
